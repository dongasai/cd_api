# 协议转换逻辑重构计划

> 创建时间: 2026-03-22
> 状态: 待批准

## 目标

修复协议转换逻辑错误，实现正确的数据流：

```
JSON → 协议结构体 → 判断转换 → 目标协议结构体 → Provider
```

## 核心变更

### 接口定义变更

| 接口 | 当前 | 修改后 |
|------|------|--------|
| `DriverInterface::parseRequest()` | 返回 `Request` (Shared\DTO) | 返回协议结构体 |
| `DriverInterface::buildResponse()` | 接收 `Response` | 接收协议响应结构体 |
| `ProviderInterface::send()` | 接收 `Request` | 接收协议结构体 |
| `ProviderInterface::sendStream()` | 接收 `Request` | 接收协议结构体 |
| `AbstractProvider::buildRequestBody()` | 接收 `Request` | 接收协议结构体 |

```php
// 修改前
public function parseRequest(array $rawRequest): Request
{
    $request = ChatCompletionRequest::fromArrayValidated($rawRequest);
    return $request->toSharedDTO();  // ❌ 返回 Shared\DTO
}

// 修改后
public function parseRequest(array $rawRequest): ChatCompletionRequest
{
    return ChatCompletionRequest::fromArrayValidated($rawRequest);  // ✅ 直接返回协议结构体
}

// 修改前
public function buildResponse(Response $response): array
{
    return ChatCompletionResponse::fromSharedDTO($response)->toArray();
}

// 修改后
public function buildResponse(ChatCompletionResponse $response): array
{
    return $response->toArray();
}
```

### 阶段 6：修改 ProtocolConverter

**文件**：`laravel/app/Services/Protocol/ProtocolConverter.php`

**核心变更**：

```php
// 新增：协议结构体转换方法
public function convert(
    ProtocolRequest $sourceRequest,
    string $targetProtocol
): ProtocolRequest {
    // 源协议 == 目标协议，直接返回
    if ($sourceRequest instanceof $this->getProtocolClass($targetProtocol)) {
        return $sourceRequest;
    }

    // 需要转换：通过 Shared\DTO 中间层
    $sharedDTO = $sourceRequest->toSharedDTO();
    $targetDriver = $this->driver($targetProtocol);

    return $targetDriver::fromSharedDTO($sharedDTO);
}

// 修改：normalizeRequest 返回协议结构体
public function normalizeRequest(array $rawRequest, string $protocol): ProtocolRequest
{
    return $this->driver($protocol)->parseRequest($rawRequest);
}

// 修改：denormalizeResponse 接收协议结构体
public function denormalizeResponse(ProtocolResponse $response, string $protocol): array
{
    return $response->toArray();
}
```
