# MCP 客户端功能实施计划

> 任务：项目增加 MCP 连接能力，安装 php-mcp/client，创建 MCP 客户端管理功能

## 上下文

- **技术栈**: Laravel 12 + PHP 8.3 + MySQL + dcat-admin v2
- **选定方案**: php-mcp/client（专注 MCP Client，轻量级）
- **用途**: 代理调用外部 MCP 工具
- **协议**: HTTP/SSE + stdio

## 实施步骤

### 步骤 1: 安装 php-mcp/client 包

**操作**:
```bash
cd laravel && composer require php-mcp/client
```

**预期结果**: vendor 目录下安装 php-mcp/client 及 ReactPHP 依赖

---

### 步骤 2: 创建数据库迁移文件

**文件**: `laravel/database/migrations/2026_04_06_XXXXXX_create_mcp_clients_table.php`

**表结构**:
```
mcp_clients
├── id (bigint unsigned, primary)
├── name (varchar(255)) - 客户端名称
├── slug (varchar(100), unique) - 标识符
├── transport (enum: 'stdio', 'http_sse') - 传输协议
├── url (varchar(500), nullable) - HTTP+SSE URL
├── command (varchar(500), nullable) - stdio 命令
├── args (json, nullable) - stdio 参数
├── headers (json, nullable) - HTTP 请求头
├── timeout (int unsigned, default 30) - 连接超时秒数
├── status (enum: 'active', 'inactive', 'error') - 状态
├── last_connected_at (timestamp, nullable) - 最后连接时间
├── connection_error (text, nullable) - 连接错误信息
├── capabilities (json, nullable) - 服务器能力列表
├── description (text, nullable) - 描述
├── created_at (timestamp)
├── updated_at (timestamp)
├── deleted_at (timestamp, nullable)
```

**预期结果**: 执行迁移后数据库新增 mcp_clients 表

### 步骤 3: 创建 McpClient 模型

**文件**: `laravel/app/Models/McpClient.php`

**功能**:
- 定义 fillable、casts、状态枚举常量
- 添加软删除 SoftDeletes trait
- 定义状态访问器

**预期结果**: 可通过 Eloquent 操作 mcp_clients 表

---

### 步骤 4: 创建 McpClientService 服务类

**文件**: `laravel/app/Services/McpClientService.php`

**功能**:
- `connect(int $clientId)` - 连接 MCP Server
- `disconnect(int $clientId)` - 断开连接
- `listTools(int $clientId)` - 获取工具列表
- `callTool(int $clientId, string $toolName, array $arguments)` - 调用工具
- `listResources(int $clientId)` - 获取资源列表
- `listPrompts(int $clientId)` - 获取提示列表
- `testConnection(int $clientId)` - 测试连接（返回 bool + 错误信息）
- 使用 php-mcp/client 的同步 API

**预期结果**: 服务类封装 MCP Client 连接和调用逻辑

---

### 步骤 5: 创建 Admin 控制器

**文件**: `laravel/app/Admin/Controllers/McpClientController.php`

**功能**:
- Grid: 列表展示（名称、协议、状态、最后连接时间）
- Form: 创建/编辑表单
- Show: 详情展示（包含 capabilities 信息）
- 自定义动作:
  - `TestConnection` - 测试连接按钮
  - `RefreshCapabilities` - 刷新服务器能力

**预期结果**: Admin 后台可管理 MCP 客户端
