# 功能开发：安装/升级网页

## 任务概述

创建安装/升级网页，通过网页进行系统安装和数据库升级。

## 实现内容

### 安装流程（7步骤）

1. **APP_KEY 检测与生成** - 自动检测并生成 APP_KEY
2. **安装锁检测** - 检测 `storage/installed.lock` 文件
3. **环境检测** - PHP版本、扩展、目录权限
4. **配置填写** - APP_URL、数据库配置、连接测试
5. **数据库迁移** - 执行 `php artisan migrate`
6. **初始化数据** - 执行 Seeder（AdminTablesSeeder、SystemSettingSeeder等）
7. **创建管理员** - 创建管理员账号并关联角色

### 升级流程

- 显示当前版本和待执行迁移列表
- 点击按钮执行迁移
- 显示执行日志

### 创建的文件

#### 控制器
- `laravel/app/Http/Controllers/InstallController.php` - 安装控制器
- `laravel/app/Http/Controllers/UpgradeController.php` - 升级控制器

#### 中间件
- `laravel/app/Http/Middleware/InstalledMiddleware.php` - 安装检测中间件

#### 服务类
- `laravel/app/Services/Install/EnvironmentChecker.php` - 环境检测服务
- `laravel/app/Services/Install/ConfigService.php` - 配置服务
- `laravel/app/Services/Install/MigrationService.php` - 迁移服务
- `laravel/app/Services/Install/AdminService.php` - 管理员服务

#### 视图
- `laravel/resources/views/install/layout.blade.php` - 安装布局
- `laravel/resources/views/install/index.blade.php` - 安装首页
- `laravel/resources/views/install/environment.blade.php` - 环境检测
- `laravel/resources/views/install/config.blade.php` - 配置填写
- `laravel/resources/views/install/migrate.blade.php` - 数据库迁移
- `laravel/resources/views/install/admin.blade.php` - 创建管理员
- `laravel/resources/views/install/complete.blade.php` - 安装完成
- `laravel/resources/views/upgrade/index.blade.php` - 升级页面

### 路由

```
/install                 → 安装首页
/install/environment     → 环境检测
/install/config          → 配置填写
/install/migrate         → 数据库迁移
/install/admin           → 创建管理员
/install/complete        → 安装完成

/upgrade                 → 升级页面
/upgrade/execute         → 执行升级
```

### 安全机制

- 文件锁机制：`storage/installed.lock`
- CSRF Token 保护
- 配置文件权限 600
