# MCP 客户端功能实现

> 2026-04-06 13:40

## 任务概述

为 CdApi 项目增加 MCP (Model Context Protocol) 客户端连接能力，支持代理调用外部 MCP Server 工具。

## 技术选型

### 库选择对比

| 库 | 说明 | 最终选择 |
|---|---|---|
| `php-mcp/client` | 专注 Client，轻量级，API 友好 | ✅ 选用 |
| `symfony/mcp-sdk` | PHP 官方 SDK，但功能冗余 | 未选用 |

**选择理由：**
- php-mcp/client 命名空间独立，与 laravel/mcp 无冲突
- 专注 Client 功能，无冗余 Server 代码
- Builder 模式 API 友好，支持同步/异步两种方式

### 与 laravel/mcp 兼容性

- `laravel/mcp` 仅提供 **Server** 功能
- `php-mcp/client` 提供 **Client** 功能
- 命名空间不同（`Laravel\Mcp` vs `PhpMcp\Client`），无冲突

    connection_error TEXT,

    description TEXT,
    created_at TIMESTAMP,
    updated_at TIMESTAMP,

## 功能特性

### 支持的传输协议

1. **HTTP+SSE** - 通过 HTTP 连接远程 MCP Server
2. **Stdio** - 通过标准输入输出连接本地 MCP Server

### 核心功能

- 连接外部 MCP Server
- 获取服务器能力（Tools、Resources、Prompts）
- 调用远程 MCP 工具
- 测试连接状态
- Admin 后台可视化管理

## 使用示例

### 创建 MCP 客户端

```php
// 通过 Admin 后台创建，或：
McpClient::create([
    'name' => 'CdApi Local',
    'slug' => 'cdapi-local',
    'transport' => 'http',
    'url' => 'http://127.0.0.1:32126/mcp/cdapi',
    'headers' => ['Authorization' => 'Bearer sk-xxx'],
    'timeout' => 30,
]);
```

### 调用 MCP 工具

```php
$service = app(McpClientService::class);
$client = McpClient::where('slug', 'cdapi-local')->first();

// 获取工具列表
$tools = $service->listTools($client);

// 调用工具
$result = $service->callTool($client, 'search', ['query' => 'API']);
```

## 已知问题

### HTTP 协议兼容性

**问题描述：**
- `laravel/mcp` Server 使用 **Streamable HTTP** 传输（POST 请求）
- `php-mcp/client` 使用 **HTTP with SSE** 传输（GET + SSE 流）

两种传输模式不兼容，导致 HTTP 方式无法连接 laravel/mcp Server。

**解决方案：**
1. 使用 **Stdio** 传输连接本地 MCP Server
2. 等待 php-mcp/client 支持 Streamable HTTP
3. 或使用其他支持 Streamable HTTP 的 MCP 客户端

**Stdio 连接示例：**
```
命令: php
参数: ["artisan", "mcp:start", "cdapi"]
```
