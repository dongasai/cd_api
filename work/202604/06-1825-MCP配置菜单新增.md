# MCP配置菜单新增

## 时间
2026-04-06 18:25

## 背景
用户要求新增顶级菜单 "MCP配置"，将 McpClient 和搜索引擎配置移动进来。

## 完成工作

### 1. 创建 SearchDriverController
- 路径: `laravel/app/Admin/Controllers/SearchDriverController.php`
- 功能: 搜索驱动配置管理
- 支持字段: name, slug, driver_class, config, timeout, priority, is_default, status, description
- 操作: 列表、详情、创建、编辑、设为默认

### 2. 创建 SetDefaultSearchDriver Action
- 路径: `laravel/app/Admin/Controllers/Actions/SetDefaultSearchDriver.php`
- 功能: 设置默认搜索驱动的行操作按钮

### 3. 添加路由
```php
// 搜索驱动配置管理
$router->resource('search-drivers', 'SearchDriverController');
```

### 4. 创建菜单结构
数据库操作：
- 创建顶级菜单 `mcp_config` (ID: 87)
- 移动 `mcp_clients` (ID: 84) 到 MCP配置 下
- 创建 `search_drivers` (ID: 88) 菜单项

### 5. 更新语言包
- 更新 `menu.php` 添加 MCP配置 菜单翻译
- 创建 `admin-search-driver.php` 搜索驱动语言包

### 6. 修复 Bug
- 修复 `Grid\Actions` 类型引用错误，改为 `Actions` (Displayers\Actions)

## 菜单结构
```
MCP配置
├── MCP 客户端
└── 搜索引擎
```
