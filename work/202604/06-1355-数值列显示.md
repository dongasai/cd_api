# Coding账户管理列表增加数值列

## 需求

后台Coding账户管理列表增加"数值"列，将数据集的数值由驱动处理后显示，不同驱动的显示方式是不同的。

## 实现方案

### 1. 修改接口

在 `CodingStatusDriver` 接口添加 `formatQuotaDisplay()` 方法：

```php
public function formatQuotaDisplay(): string;
```

### 2. 抽象基类默认实现

在 `AbstractCodingStatusDriver` 中添加默认实现：
- 显示各维度的已用/总量百分比
- 根据使用率选择颜色（success/info/warning/danger）
- 格式化大数字（K, M单位）

### 3. 各驱动特定实现

| 驱动 | 显示格式 |
|------|----------|
| RequestCodingStatusDriver | 请求次数: {used}/{limit} |
| TokenCodingStatusDriver | Token: {used}/{limit} (K/M单位) |
| PromptCodingStatusDriver | Prompt次数: {used}/{limit} |
| GLMCodingStatusDriver | Prompt: {used}/{limit}<br>Token: {used}/{limit} |
| Request5ZMCodingStatusDriver | 5h: {used}/{limit}<br>周: {used}/{limit}<br>月: {used}/{limit} |
| SlidingRequestCodingStatusDriver | {used}/{limit} (窗口类型) |
| SlidingTokenCodingStatusDriver | 总计: {used}/{limit} (窗口类型)<br>输入/输出明细 |

### 4. 控制器修改

在 `CodingAccountController` 的 `grid()` 方法中添加数值列：

```php
$grid->column('quota_display', '数值')->unescape()->display(function () {
    try {
        $driver = app(CodingStatusDriverManager::class)->driver($this->driver_class);
        $driver->setAccount($this);
        return $driver->formatQuotaDisplay();
    } catch (\Exception $e) {
        return '<span class="text-muted">-</span>';
    }
});
```
