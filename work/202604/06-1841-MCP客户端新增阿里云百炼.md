## MCP 连接问题修复

**问题**：SDK 的 `ClientCapabilities` 空对象序列化为 `[]`，阿里云服务器期望对象格式 `{}`。

**解决方案**：修改 `McpClientService::createClient()`，设置 `roots: true` 能力：
```php
$capabilities = new ClientCapabilities(roots: true);
$builder->setCapabilities($capabilities);
```

## 默认驱动问题修复

**问题**：`SearchServiceProvider` 在注册单例后，用配置文件的默认值覆盖了数据库设置。

**解决方案**：移除 ServiceProvider 中的配置文件默认驱动设置，让 `SearchDriverManager` 从数据库自动加载。
