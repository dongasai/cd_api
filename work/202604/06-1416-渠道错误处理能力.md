# CodingStatus 渠道错误处理能力

## 需求

CodingStatus 增加渠道错误处理能力：当渠道请求错误时，根据返回的状态码和内容匹配规则，自动暂停 CodingAccount 并按配置时间自动恢复。

## 实现方案

### 核心设计

```
┌─────────────────────────────────────────────────────────────────────┐
│                        错误处理流程                                  │
├─────────────────────────────────────────────────────────────────────┤
│  请求失败 → ChannelRequestLog → 匹配 ChannelErrorRule              │
│                                     ↓                               │
│                              暂停 CodingAccount                     │
│                                     ↓                               │
│         status=suspended, disabled_at=now, pause_duration_minutes  │
│                                     ↓                               │
│                        所有关联渠道不可选（路由过滤）                │
│                                     ↓                               │
│              AutoReopenCodingAccounts 检查恢复                      │
│         (disabled_at + pause_duration_minutes 已过期？)             │
└─────────────────────────────────────────────────────────────────────┘
```

### 字段复用

| 字段 | 用途 | 来源 |
|------|------|------|
| `status` | 账户状态 | 已有 |
| `disabled_at` | 暂停开始时间 | 已有 |
| `pause_duration_minutes` | 暂停时长 | **新增** |
| `pause_reason` | 暂停原因 | **新增** |
| `pause_rule_id` | 触发规则ID | **新增** |

### 恢复机制

修改现有 `AutoReopenCodingAccounts` 命令，支持两种模式：
- 有 `pause_duration_minutes` → 用它计算恢复时间
- 无 `pause_duration_minutes` → 用原有 `auto_reopen_hours`

## 规则配置示例

| 规则名称 | 匹配类型 | 匹配值 | 暂停时长 |
|---------|---------|--------|---------|
| HTTP 429 限流 | status_code | 429 | 10分钟 |
| HTTP 401 认证失败 | status_code | 401 | 60分钟 |
| 达到5小时限额 | error_message | 达到5小时限额 | 1小时 |
| 达到周限额 | error_message | 达到周限额 | 1天 |

## 使用方式

### 后台管理

访问 `/admin/error-rules` 管理错误处理规则。

### 手动暂停/恢复

```php
use App\Services\CodingStatus\ChannelErrorHandlingService;

// 手动暂停账户
$errorHandlingService->manualPauseAccount($account, 30, $userId, '测试暂停');

// 手动恢复账户
$errorHandlingService->manualRecoverAccount($account, $userId);
```

### 定时恢复

现有 `cdapi:coding:auto-reopen` 命令自动处理恢复。
