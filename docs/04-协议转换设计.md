# 协议转换设计

## 一、协议对比

### 1.1 OpenAI Chat Completions API

```
┌─────────────────────────────────────────────────────────────────────┐
│                    OpenAI Chat Completions API                       │
└─────────────────────────────────────────────────────────────────────┘

端点: POST /v1/chat/completions

请求格式:
{
    "model": "gpt-4",
    "messages": [
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Hello!"},
        {"role": "assistant", "content": "Hi! How can I help?"},
        {"role": "user", "content": "Tell me a joke."}
    ],
    "temperature": 0.7,
    "max_tokens": 100,
    "top_p": 1.0,
    "n": 1,
    "stream": false,
    "stop": ["\n", "END"],
    "presence_penalty": 0.0,
    "frequency_penalty": 0.0,
    "logit_bias": {"50256": -100},
    "user": "user-123"
}

响应格式:
{
    "id": "chatcmpl-abc123",
    "object": "chat.completion",
    "created": 1677858242,
    "model": "gpt-4",
    "choices": [
        {
            "index": 0,
            "message": {
                "role": "assistant",
                "content": "Why did the chicken cross the road?"
            },
            "finish_reason": "stop"
        }
    ],
    "usage": {
        "prompt_tokens": 20,
        "completion_tokens": 10,
        "total_tokens": 30
    }
}

流式响应 (SSE):
data: {"id":"chatcmpl-abc123","object":"chat.completion.chunk","created":1677858242,"model":"gpt-4","choices":[{"index":0,"delta":{"role":"assistant"},"finish_reason":null}]}

data: {"id":"chatcmpl-abc123","object":"chat.completion.chunk","created":1677858242,"model":"gpt-4","choices":[{"index":0,"delta":{"content":"Why"},"finish_reason":null}]}

data: {"id":"chatcmpl-abc123","object":"chat.completion.chunk","created":1677858242,"model":"gpt-4","choices":[{"index":0,"delta":{"content":" did"},"finish_reason":null}]}

data: {"id":"chatcmpl-abc123","object":"chat.completion.chunk","created":1677858242,"model":"gpt-4","choices":[{"index":0,"delta":{},"finish_reason":"stop"}]}

data: [DONE]
```

### 1.2 Anthropic Messages API

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Anthropic Messages API                            │
└─────────────────────────────────────────────────────────────────────┘

端点: POST /v1/messages

请求头:
x-api-key: {api_key}
anthropic-version: 2023-06-01

请求格式:
{
    "model": "claude-3-opus-20240229",
    "system": "You are a helpful assistant.",
    "messages": [
        {"role": "user", "content": "Hello!"},
        {"role": "assistant", "content": "Hi! How can I help?"},
        {"role": "user", "content": "Tell me a joke."}
    ],
    "max_tokens": 100,
    "temperature": 0.7,
    "top_p": 1.0,
    "top_k": 40,
    "stream": false,
    "stop_sequences": ["\n", "END"],
    "metadata": {
        "user_id": "user-123"
    }
}

响应格式:
{
    "id": "msg_abc123",
    "type": "message",
    "role": "assistant",
    "content": [
        {
            "type": "text",
            "text": "Why did the chicken cross the road?"
        }
    ],
    "model": "claude-3-opus-20240229",
    "stop_reason": "end_turn",
    "stop_sequence": null,
    "usage": {
        "input_tokens": 20,
        "output_tokens": 10
    }
}

流式响应 (SSE):
event: message_start
data: {"type":"message_start","message":{"id":"msg_abc123","type":"message","role":"assistant","content":[],"model":"claude-3-opus-20240229","stop_reason":null,"usage":{"input_tokens":20,"output_tokens":0}}}

event: content_block_start
data: {"type":"content_block_start","index":0,"content_block":{"type":"text","text":""}}

event: content_block_delta
data: {"type":"content_block_delta","index":0,"delta":{"type":"text_delta","text":"Why"}}

event: content_block_delta
data: {"type":"content_block_delta","index":0,"delta":{"type":"text_delta","text":" did"}}

event: content_block_stop
data: {"type":"content_block_stop","index":0}

event: message_delta
data: {"type":"message_delta","delta":{"stop_reason":"end_turn"},"usage":{"output_tokens":10}}

event: message_stop
data: {"type":"message_stop"}
```

## 二、字段映射表

### 2.1 请求字段映射

```
┌─────────────────────────────────────────────────────────────────────┐
│                    OpenAI → Anthropic 请求转换                       │
└─────────────────────────────────────────────────────────────────────┘

┌──────────────────────┬────────────────────────────────────────────┐
│ OpenAI 字段          │ Anthropic 字段                             │
├──────────────────────┼────────────────────────────────────────────┤
│ model                │ model                                      │
│ messages[].role=     │ system (顶层字段)                          │
│   "system"           │                                            │
│ messages[] (其他)    │ messages[] (保持顺序，移除 system)         │
│ max_tokens           │ max_tokens                                 │
│ temperature          │ temperature                                │
│ top_p                │ top_p                                      │
│ stop                 │ stop_sequences                             │
│ stream               │ stream                                     │
│ user                 │ metadata.user_id                           │
│ presence_penalty     │ (不支持，忽略)                             │
│ frequency_penalty    │ (不支持，忽略)                             │
│ logit_bias           │ (不支持，忽略)                             │
│ n                    │ (不支持，Anthropic 只返回一个响应)         │
└──────────────────────┴────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│                    Anthropic → OpenAI 请求转换                       │
└─────────────────────────────────────────────────────────────────────┘

┌──────────────────────┬────────────────────────────────────────────┐
│ Anthropic 字段       │ OpenAI 字段                                │
├──────────────────────┼────────────────────────────────────────────┤
│ model                │ model                                      │
│ system               │ messages.unshift({role: "system", ...})    │
│ messages[]           │ messages[]                                 │
│ max_tokens           │ max_tokens                                 │
│ temperature          │ temperature                                │
│ top_p                │ top_p                                      │
│ top_k                │ (不支持，忽略)                             │
│ stop_sequences       │ stop                                       │
│ stream               │ stream                                     │
│ metadata.user_id     │ user                                       │
└──────────────────────┴────────────────────────────────────────────┘
```

### 2.2 响应字段映射

```
┌─────────────────────────────────────────────────────────────────────┐
│                    OpenAI → Anthropic 响应转换                       │
└─────────────────────────────────────────────────────────────────────┘

┌──────────────────────┬────────────────────────────────────────────┐
│ OpenAI 字段          │ Anthropic 字段                             │
├──────────────────────┼────────────────────────────────────────────┤
│ id                   │ id                                         │
│ object               │ type: "message"                            │
│ model                │ model                                      │
│ choices[].message    │ content: [{type: "text", text: ...}]      │
│ choices[].finish_    │ stop_reason                                │
│   reason             │                                            │
│   "stop"             │   → "end_turn"                             │
│   "length"           │   → "max_tokens"                           │
│   "content_filter"   │   → "stop_sequence"                        │
│ usage.prompt_tokens  │ usage.input_tokens                         │
│ usage.completion_    │ usage.output_tokens                        │
│   tokens             │                                            │
│ usage.total_tokens   │ (计算: input + output)                     │
└──────────────────────┴────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│                    Anthropic → OpenAI 响应转换                       │
└─────────────────────────────────────────────────────────────────────┘

┌──────────────────────┬────────────────────────────────────────────┐
│ Anthropic 字段       │ OpenAI 字段                                │
├──────────────────────┼────────────────────────────────────────────┤
│ id                   │ id                                         │
│ type: "message"      │ object: "chat.completion"                  │
│ model                │ model                                      │
│ content[].text       │ choices[].message.content                  │
│ stop_reason          │ choices[].finish_reason                    │
│   "end_turn"         │   → "stop"                                 │
│   "max_tokens"       │   → "length"                               │
│   "stop_sequence"    │   → "stop"                                 │
│ usage.input_tokens   │ usage.prompt_tokens                        │
│ usage.output_tokens  │ usage.completion_tokens                    │
│ (计算)               │ usage.total_tokens                         │
│ role: "assistant"    │ choices[].message.role                     │
└──────────────────────┴────────────────────────────────────────────┘
```

## 三、转换器设计

### 3.1 类设计

```
┌─────────────────────────────────────────────────────────────────────┐
│                     ProtocolTransformer                              │
├─────────────────────────────────────────────────────────────────────┤
│ + transformRequest(from: string, to: string, data: array): array    │
│ + transformResponse(from: string, to: string, data: array): array   │
│ + transformStreamChunk(from: string, to: string, chunk: string)     │
│ + getTransformer(from: string, to: string): TransformerInterface    │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│                     TransformerInterface                             │
├─────────────────────────────────────────────────────────────────────┤

┌─────────────────────────────────────────────────────────────────────┐
│                     OpenAIToAnthropic                                │
├─────────────────────────────────────────────────────────────────────┤
│ + transformRequest(data: array): array                              │
│ + transformResponse(data: array): array                             │
│ + transformStreamChunk(chunk: string): string                       │

┌─────────────────────────────────────────────────────────────────────┐
│                     AnthropicToOpenAI                                │

│ - injectSystemMessage(messages: array, system: string): array       │

└─────────────────────────────────────────────────────────────────────┘
```

## 四、流式响应转换

### 4.1 OpenAI 流式格式

```
事件格式: data: {json}\n\n

消息开始:
data: {"id":"chatcmpl-xxx","object":"chat.completion.chunk","created":123,"model":"gpt-4","choices":[{"index":0,"delta":{"role":"assistant"},"finish_reason":null}]}

内容增量:
data: {"id":"chatcmpl-xxx","object":"chat.completion.chunk","created":123,"model":"gpt-4","choices":[{"index":0,"delta":{"content":"Why"},"finish_reason":null}]}

结束:
data: {"id":"chatcmpl-xxx","object":"chat.completion.chunk","created":123,"model":"gpt-4","choices":[{"index":0,"delta":{},"finish_reason":"stop"}]}

完成:
data: [DONE]
```

### 4.2 Anthropic 流式格式

```
事件格式: event: {event_type}\ndata: {json}\n\n

消息开始:
event: message_start
data: {"type":"message_start","message":{...}}

内容块开始:
event: content_block_start
data: {"type":"content_block_start","index":0,"content_block":{"type":"text","text":""}}

内容增量:
event: content_block_delta
data: {"type":"content_block_delta","index":0,"delta":{"type":"text_delta","text":"Why"}}

内容块结束:
event: content_block_stop
data: {"type":"content_block_stop","index":0}

消息增量:
event: message_delta
data: {"type":"message_delta","delta":{"stop_reason":"end_turn"},"usage":{...}}

消息结束:
event: message_stop
data: {"type":"message_stop"}
```

### 4.3 流式转换实现

```
┌─────────────────────────────────────────────────────────────────────┐
│                   Anthropic → OpenAI 流式转换                        │
└─────────────────────────────────────────────────────────────────────┘

状态机:
┌─────────────┐     message_start      ┌─────────────┐
│   IDLE      │ ──────────────────────▶│  STARTED    │
└─────────────┘                        └──────┬──────┘
                                              │
                              content_block_  │
                              start           │
                                              ▼
                                       ┌─────────────┐
                                       │  STREAMING  │
                                       └──────┬──────┘
                                              │
                              content_block_  │
                              delta           │
                                              ▼
                                       ┌─────────────┐
                              ┌────────│  DELTA      │
                              │        └─────────────┘
                              │              │
                              │              │ content_block_stop
                              │              ▼
                              │       ┌─────────────┐
                              └───────│  FLUSH      │
                                      └──────┬──────┘
                                             │
                              message_delta  │
                                             ▼
                                      ┌─────────────┐
                                      │  FINISH     │
                                      └──────┬──────┘
                                             │
                              message_stop   │
                                             ▼
                                      ┌─────────────┐
                                      │  DONE       │
                                      └─────────────┘

转换逻辑:
1. message_start → 生成 OpenAI 格式的初始 chunk
2. content_block_delta → 累积文本，生成 OpenAI delta chunk
3. message_delta → 生成 finish_reason
4. message_stop → 发送 [DONE]
```

## 五、特殊处理

### 5.1 多模态内容

```
┌─────────────────────────────────────────────────────────────────────┐
│                        多模态内容处理                                │
└─────────────────────────────────────────────────────────────────────┘

OpenAI 格式 (图片):
{
    "messages": [{
        "role": "user",
        "content": [
            {"type": "text", "text": "What's in this image?"},
            {
                "type": "image_url",
                "image_url": {
                    "url": "https://example.com/image.png",
                    "detail": "high"
                }
            }
        ]
    }]
}

Anthropic 格式 (图片):
{
    "messages": [{
        "role": "user",
        "content": [
            {"type": "text", "text": "What's in this image?"},
            {
                "type": "image",
                "source": {
                    "type": "url",
                    "url": "https://example.com/image.png"
                }
            }
        ]
    }]
}

转换规则:
┌──────────────────────────┬────────────────────────────────────────┐
│ OpenAI                   │ Anthropic                              │
├──────────────────────────┼────────────────────────────────────────┤
│ type: "image_url"        │ type: "image"                          │
│ image_url.url            │ source.type = "url", source.url        │
│ image_url.detail         │ (忽略，Anthropic 不支持)               │
│ image_url.url (base64)   │ source.type = "base64",                │
│                          │ source.media_type, source.data         │
└──────────────────────────┴────────────────────────────────────────┘
```

### 5.2 工具调用 (Function Calling)

```
┌─────────────────────────────────────────────────────────────────────┐
│                        工具调用处理                                  │
└─────────────────────────────────────────────────────────────────────┘

OpenAI 格式:
{
    "tools": [{
        "type": "function",
        "function": {
            "name": "get_weather",
            "description": "Get weather info",
            "parameters": {...}
        }
    }],
    "tool_choice": "auto"
}

响应:
{
    "choices": [{
        "message": {
            "role": "assistant",
            "content": null,
            "tool_calls": [{
                "id": "call_xxx",
                "type": "function",
                "function": {
                    "name": "get_weather",
                    "arguments": "{\"location\": \"Beijing\"}"
                }
            }]
        }
    }]
}

Anthropic 格式:
{
    "tools": [{
        "name": "get_weather",
        "description": "Get weather info",
        "input_schema": {...}
    }]
}

响应:
{
    "content": [{
        "type": "tool_use",
        "id": "toolu_xxx",
        "name": "get_weather",
        "input": {"location": "Beijing"}
    }]
}

转换规则:
┌──────────────────────────┬────────────────────────────────────────┐
│ OpenAI                   │ Anthropic                              │
├──────────────────────────┼────────────────────────────────────────┤
│ tools[].type: "function" │ tools[] (无 type 字段)                 │
│ function.name            │ name                                   │
│ function.description     │ description                            │
│ function.parameters      │ input_schema                           │
│ tool_choice: "auto"      │ (默认行为)                             │
│ tool_choice: "none"      │ (不支持)                               │
│ tool_choice: "required"  │ (不支持)                               │
│ tool_calls[].id          │ content[].id                           │
│ tool_calls[].function    │ content[].name + input                 │
│ function.arguments (str) │ input (object)                         │
└──────────────────────────┴────────────────────────────────────────┘
```

## 六、模型映射

```

### 6.2 跨 Provider 转换

```
┌─────────────────────────────────────────────────────────────────────┐
│                    跨 Provider 模型转换                              │
└─────────────────────────────────────────────────────────────────────┘

场景: 客户端请求 OpenAI 格式，但使用 Anthropic 渠道

请求:
POST /v1/chat/completions
{
    "model": "gpt-4",
    "messages": [...]
}

处理流程:
1. 模型映射: "gpt-4" → "claude-3-opus-20240229"
2. 选择渠道: Anthropic 渠道
3. 协议转换: OpenAI → Anthropic 请求格式
4. 发送请求: 到 Anthropic API
5. 协议转换: Anthropic → OpenAI 响应格式
6. 返回响应: OpenAI 格式给客户端

注意事项:
- 不是所有模型都支持跨 Provider 转换
- 需要考虑上下文长度差异
- 需要考虑功能差异 (如 function calling)
- Token 计数可能不准确
```

### 7.5 Token 估算差异

```
┌─────────────────────────────────────────────────────────────────────┐
│                      Token 估算差异处理                              │
└─────────────────────────────────────────────────────────────────────┘

不同 Provider 的 Token 计算差异:
┌─────────────────┬──────────────────────────────────────────────────┐
│ Provider        │ Tokenizer                                        │
├─────────────────┼──────────────────────────────────────────────────┤
│ OpenAI          │ tiktoken (cl100k_base for GPT-4)                 │
│ Anthropic       │ 内部 Tokenizer (近似 GPT-4)                      │
│ Google Gemini   │ 内部 Tokenizer                                   │
└─────────────────┴──────────────────────────────────────────────────┘

估算策略:
┌─────────────────┬──────────────────────────────────────────────────┐
│ 场景            │ 处理方式                                         │
├─────────────────┼──────────────────────────────────────────────────┤
│ 请求前估算      │ 使用源模型 Tokenizer 估算                        │
│ 请求后实际      │ 使用响应中的 usage 字段                          │
│ 差异处理        │ 记录差异，用于后续优化估算                       │
│ 配额计算        │ 以实际 usage 为准                                │
└─────────────────┴──────────────────────────────────────────────────┘

Token 估算修正系数:
┌─────────────────────┬──────────────────────────────────────────────┐
│ 源 → 目标           │ 修正系数 (目标/源)                            │
├─────────────────────┼──────────────────────────────────────────────┤
│ OpenAI → Anthropic  │ 1.0 - 1.1 (Anthropic 略多)                   │
│ Anthropic → OpenAI  │ 0.9 - 1.0                                    │
│ OpenAI → Gemini     │ 1.0 - 1.2                                    │
│ Gemini → OpenAI     │ 0.8 - 1.0                                    │
└─────────────────────┴──────────────────────────────────────────────┘

实现示例:
function estimateTokens(messages, sourceProvider, targetProvider) {
    // 使用源 Provider 的 Tokenizer 估算
    let estimatedTokens = sourceProvider.countTokens(messages);
    
    // 应用修正系数
    const factor = getCorrectionFactor(sourceProvider, targetProvider);
    estimatedTokens = Math.ceil(estimatedTokens * factor);
    
    return estimatedTokens;
}
