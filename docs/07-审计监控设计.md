## 三、成本计算

### 3.1 定价模型

```
┌─────────────────────────────────────────────────────────────────────┐
│                        模型定价配置                                  │
└─────────────────────────────────────────────────────────────────────┘

定价配置:
{
    "gpt-4-turbo": {
        "input_price": 0.01,      // 美元/1K tokens
        "output_price": 0.03,     // 美元/1K tokens
        "currency": "USD"
    },
    "gpt-3.5-turbo": {
        "input_price": 0.0005,
        "output_price": 0.0015,
        "currency": "USD"
    },
    "claude-3-opus": {
        "input_price": 0.015,
        "output_price": 0.075,
        "currency": "USD"
    },
    "claude-3-sonnet": {
        "input_price": 0.003,
        "output_price": 0.015,
        "currency": "USD"
    }
}

成本计算公式:
cost = (input_tokens / 1000 * input_price) + (output_tokens / 1000 * output_price)

示例:
输入: 1000 tokens, 输出: 500 tokens, 模型: gpt-4-turbo
cost = (1000/1000 * 0.01) + (500/1000 * 0.03)
     = 0.01 + 0.015
     = 0.025 USD
```

### 3.2 成本计算服务

```
┌─────────────────────────────────────────────────────────────────────┐
│                        CostCalculator                                │
├─────────────────────────────────────────────────────────────────────┤
│ + calculate(model: string, inputTokens: int, outputTokens: int)     │
│ + getPrice(model: string): PriceInfo                                │
│ + updatePrice(model: string, price: PriceInfo): void                │
│ - pricingConfig: array                                              │
└─────────────────────────────────────────────────────────────────────┘

使用示例:
$calculator = new CostCalculator();
$cost = $calculator->calculate('gpt-4-turbo', 1000, 500);
// $cost = 0.025

价格更新:
├─── 从配置文件加载
├─── 从数据库加载
├─── 从外部 API 获取 (实时价格)
└─── 支持手动更新
```

```
return [

        'php_info',
```

### 5.2 告警规则

```
┌─────────────────────────────────────────────────────────────────────┐
│                        告警规则配置                                  │
└─────────────────────────────────────────────────────────────────────┘

高错误率告警:
ALERT HighErrorRate
  IF sum(rate(cdapi_requests_total{status=~"5.."}[5m])) 
     / sum(rate(cdapi_requests_total[5m])) > 0.05
  FOR 5m
  LABELS { severity: "critical" }
  ANNOTATIONS {
    summary: "高错误率告警",
    description: "错误率超过 5%，当前: {{ $value | humanizePercentage }}"
  }

高延迟告警:
ALERT HighLatency
  IF histogram_quantile(0.95, 
      sum(rate(cdapi_request_duration_seconds_bucket[5m])) by (le)
  ) > 5
  FOR 5m
  LABELS { severity: "warning" }
  ANNOTATIONS {
    summary: "高延迟告警",
    description: "P95 延迟超过 5 秒"

配额即将耗尽告警:
```
