# CdApi 功能特色

## 项目概述

**CdApi** - AI 大模型 API 网关工具，提供统一网关、智能路由、协议转换和全面管理。

- **技术栈**: Laravel 12.53 + PHP 8.3 + MySQL 8.0 + Dcat Admin
- **访问地址**: http://192.168.4.107:32126

---

## 核心特色功能

### 🛡️ Coding Plan（核心特色）
- 🔥 **熔断机制** - 渠道绑定Coding账户，配额耗尽自动禁用，防止超额扣费
- 🔄 **自动恢复** - 配额周期重置后自动启用渠道，无需人工干预
- 📊 **多维配额监控** - 支持5小时/周/月三维配额，Token/请求/Prompt多种计费模式
- 🎯 **智能状态驱动** - 6种状态驱动（Token/Request/Prompt/GLM/SlidingWindow/5ZM），适配不同平台
- 💰 **成本优化** - 自动避开超限渠道，最大化利用免费配额

### 🔄 统一大模型网关
- 🌐 **多提供商统一接入** - 支持 8+ AI 服务商（OpenAI、Anthropic、Azure、DeepSeek、智谱、Moonshot、Ollama）
- 🔄 **协议自动转换** - OpenAI ↔ Anthropic 双向转换（请求/响应/流式事件）
- 🔀 **智能路由分发** - 加权轮询/最少连接/响应时间优先，故障自动切换
- 🏷️ **模型别名扩展** - 一模型多名，最大化渠道匹配，提升可用性 200%
- 🎯 **渠道亲和性** - 智能粘滞选路，会话保持，支持 CLI 工具连续对话

### 📦 渠道管理
- 🏗️ **三级渠道继承** - Merge/Override/Extend 模式，配置复用与定制
- 📂 **分组标签系统** - 多维度分类（用途/地域/模型），灵活管理
- ⚙️ **动态状态管理** - Active/Disabled/Maintenance，实时控制路由
- 🔐 **User-Agent限制** - 渠道可配置允许的User-Agent规则，精准控制访问来源

### 🔐 安全与认证
- 🔑 **API Key 认证** - bcrypt 安全存储，过期管理，撤销机制
- 🎯 **Key级模型映射** - 每个API Key可配置独立模型别名映射，实现Key级别路由控制
- 💰 **金额配额** - 金额限制与请求次数限制，自动重置

### ⚡ 流量控制
- 🛡️ **限流保护** - 滑动窗口/令牌桶/漏桶算法，防止过载
- 📊 **多维限制** - 请求次数 + 并发请求，稳定服务
- 🤝 **客户端友好** - 标准限流响应，Retry-After 指引

### 📊 监控审计
- 📝 **三重日志** - 审计日志 + 请求日志 + 响应日志，完整追溯
- 📈 **实时统计** - 请求数/成功率/延迟/成本，实时监控
- 🔍 **渠道健康** - 熔断状态/失败率/性能指标，一目了然
- 💰 **成本分析** - 按模型/渠道多维度成本统计

### 🎨 管理后台
- 🖥️ **可视化配置** - 渠道/API Key/权限/限流，Web 端配置
- 🌐 **多语言支持** - 中英文切换，国际化友好
- 🧪 **内置测试** - 模型测试工具，请求回放，快速验证
- 📊 **数据看板** - 图表统计，实时监控，运维友好

### 🚀 性能与可靠
- 💾 **三级缓存** - 应用内存→Redis→数据库，命中率 >95%
- ⚡ **低延迟网关转发** - 平均延迟 <10ms，P99 <50ms，吞吐量 10,000+ RPM
- 🔄 **高可用设计** - 多渠道冗余、自动故障切换、弹性伸缩

### 🛠️ 开发者友好
- 🔧 **CLI 工具** - 渠道测试、请求回放、配额同步命令
- 📖 **OpenAPI 规范** - 标准化 API 文档，易于集成
- 🔌 **驱动架构** - Provider Driver 模式，易扩展新服务商
