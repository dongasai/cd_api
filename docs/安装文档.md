# CdApi 安装文档

## 系统要求

- Docker 20.10+
- Docker Compose 2.0+
- 内存: 建议 2GB+
- 磁盘: 建议 10GB+

## 快速开始

### 1. 克隆项目

```bash
git clone <项目地址>
cd coding_api
```

### 2. 构建并启动容器

```bash
docker-compose up -d --build
```

### 3. 访问应用

- Web 应用: http://localhost
- 后台管理: http://admin 或 http://localhost/admin

## 详细安装步骤

### 环境配置

1. **复制环境变量文件**

```bash
cd laravel
cp .env.example .env
```

2. **修改配置** (根据需要)

编辑 `.env` 文件：

```env
APP_NAME=CdApi
APP_URL=http://localhost

# 数据库配置 (默认 SQLite)
DB_CONNECTION=sqlite
DB_DATABASE=/var/www/html/database/database.sqlite

# 其他配置...
```

### Docker 部署

#### 开发环境

```bash
# 构建镜像
docker-compose build

# 启动服务
docker-compose up -d

# 查看日志
docker-compose logs -f

# 停止服务
docker-compose down
```

#### 生产环境

```bash
# 使用生产配置
docker-compose -f docker-compose.prod.yml up -d --build
```

### 初始化数据

```bash
# 进入容器
docker exec -it coding_api bash

# 生成应用密钥
php artisan key:generate

# 运行迁移
php artisan migrate

# 创建管理员账号 (可选)
php artisan make:filament-user
```

## 常用命令

### 容器管理

```bash
# 查看运行中的容器
docker-compose ps

# 重启服务
docker-compose restart

# 重建并启动
docker-compose up -d --build --force-recreate

# 查看容器日志
docker-compose logs -f app
```

### Laravel 命令

```bash
# 进入容器执行命令
docker exec -it coding_api bash

# 清除缓存
php artisan cache:clear
php artisan config:clear
php artisan view:clear

# 运行测试
php artisan test

# 队列处理
php artisan queue:work

# 定时任务
php artisan schedule:run
```

## 服务说明

容器内使用 Supervisor 管理以下进程：

| 服务 | 说明 |
|------|------|
| apache2 | Web 服务器 (Apache + PHP 8.3) |
| queue-worker | Laravel 队列处理 |
| schedule-worker | Laravel 定时任务调度 |
| init-permissions | 初始化文件权限 (启动时运行一次) |
| build-cache | 构建应用缓存 (启动时运行一次) |

## 故障排查

### 端口冲突

如果 80 端口被占用，修改 `docker-compose.yml`：

```yaml
ports:
  - "8080:80"  # 将主机 8080 映射到容器 80
```

### 权限问题

```bash
# 修复存储目录权限
docker exec -it coding_api bash
chown -R php:php /var/www/html/storage
chmod -R 775 /var/www/html/storage
```

### 查看错误日志

```bash
# Supervisor 日志
docker exec coding_api cat /var/log/supervisor/supervisord.log

# Apache 错误日志
docker exec coding_api cat /var/log/apache2/error.log

# 应用日志
docker exec coding_api cat /var/www/html/storage/logs/laravel.log
```

## 生产环境注意事项

1. **修改默认密码**: 立即修改所有默认账号密码
2. **HTTPS**: 配置 SSL 证书
3. **备份**: 定期备份数据库和配置文件
4. **监控**: 配置日志监控和告警
5. **防火墙**: 限制不必要的端口访问
