# 协议转化

> 采用由统一Dto为中间格式的转换方式，不存在A2O/O2A的方式


## 架构
1. SharedDto 统一格式（协议转换的中间格式，laravel/app/Services/Shared）
2. OpenAi实体，OpenAi输入/OpenAi返回的内容转实体（laravel/app/Services/Protocol/Driver/OpenAI）
3. Anthropic实体，Anthropic输入/Anthropic返回的内容转Anthropic实体 （laravel/app/Services/Protocol/Driver/Anthropic）
4. 不能直接转换，必须经过SharedDto作为中间

## 问题
1. toOpenai ； toAnthropic 类似的方法都是错误的
2. SharedDto 是统一格式，不应具备转换能力
3. OpenAi实体/Anthropic实体 ，应该有 fromSharedDto 的方法
