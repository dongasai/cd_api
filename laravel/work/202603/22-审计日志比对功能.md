# 审计日志比对功能

## 时间
2026-03-22 22:00

## 需求
在后台审计日志列表页面增加"比对"按钮，点击后在新标签页打开比对页面。

## 实现

### 1. 创建 Action 类
- 文件：[CompareRequestDiff.php](../laravel/app/Admin/Actions/CompareRequestDiff.php)
- 功能：在审计日志列表添加"比对"行操作按钮
- 特性：仅在存在渠道请求日志时显示，点击后在新标签页打开

### 2. 创建控制器
- 文件：[RequestDiffController.php](../laravel/app/Admin/Controllers/RequestDiffController.php)
- 功能：处理比对页面请求，解析请求数据并传递给视图

### 3. 创建视图
- 文件：[show.blade.php](../laravel/resources/views/admin/request-diff/show.blade.php)
- 功能：展示请求差异比对结果
- 包含：
  - 基本信息表格
  - 原始 Body 逐行比对（带上下文）
  - Messages 对比
  - 其他字段对比

### 4. 添加路由
- 路由：`GET admin/request-diff/{audit_log_id}`
- 名称：`dcat.admin.request-diff`

### 5. 添加翻译
- 中文：`compare_request_diff` => `比对`
- 英文：`compare_request_diff` => `Compare`

## 使用
1. 进入后台 -> 审计日志
2. 点击操作列的"比对"按钮
3. 新标签页打开比对页面，显示请求日志和渠道日志的差异
